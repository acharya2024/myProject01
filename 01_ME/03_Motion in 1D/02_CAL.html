Calculus Applications to Kinematics

Using Derivatives and Integrals in Motion Analysis.

Kinematics is the study of motion. Calculus provides powerful tools to analyze and understand motion, especially when dealing with quantities that are constantly changing, like velocity and acceleration. Differentiation, in particular, helps us determine these instantaneous rates of change.

Instantaneous Velocity:

Imagine an object moving along a straight line. Its position is constantly changing with time. The *average* velocity tells us how much the position changed over a certain time *interval*. But what if we want to know how fast the object is moving at a *specific instant*? That's where instantaneous velocity comes in.

Mathematically, instantaneous velocity (\(v\)) is defined as the derivative of the object's position (\(x\)) with respect to time (\(t\)). This means we're looking at the rate of change of position as the time interval becomes infinitesimally small:

\( \qquad v = \dfrac{dx}{dt} \)

Here, \(dx/dt\) represents the derivative of the position function \(x(t)\) with respect to time \(t\). It tells us exactly how fast the position is changing at any given moment.

Beyond Motion: Other Applications of Differentiation:

Differentiation isn't just limited to kinematics. It's a fundamental concept applicable to any situation where you want to understand how one quantity changes in relation to another. Here are a few examples:

  • Rate of Increase of Temperature: If you have a function that describes how the temperature of an object changes over time, you can use differentiation to find the rate at which the temperature is increasing or decreasing at a particular moment. This is useful in many areas, such as engineering, chemistry, and meteorology.
  • Rate of Change of Chemical Reaction: In chemistry, differentiation can be used to determine how quickly the concentration of a reactant or product is changing during a chemical reaction.
  • Population Growth: If you have a mathematical model that describes how a population is growing, you can use differentiation to find the rate at which the population is increasing or decreasing at any given time.
  • In Economics: Marginal cost, marginal revenue, and other marginal concepts are all based on differentiation, helping businesses understand how costs, revenue, and profit change with respect to production levels.

Graphical Interpretation: Unveiling Function Behavior

The derivative has a powerful visual interpretation. When you plot a function on a graph, the derivative at any point represents the slope of the line that is tangent to the curve at that point. This slope provides crucial information about the function's behavior:

  • Zero Derivative: Flatlining Moments If the derivative at a point is zero, it means the tangent line is horizontal. This indicates a *stationary point* on the curve. Stationary points can be:
    • Maximum: The function reaches a peak.
    • Minimum: The function reaches a valley.
    • Point of Inflection: The function changes its curvature.
  • Positive Derivative: The Ascent A positive derivative signifies that the function is *increasing* at that point. As you move along the curve in the positive x-direction, the y-values are getting larger. The steeper the slope (larger derivative), the faster the function is increasing.
  • Negative Derivative: The Descent Conversely, a negative derivative means the function is *decreasing*. Moving along the curve in the positive x-direction results in smaller y-values. Again, the magnitude of the slope (derivative) indicates the rate of decrease.

Illustrative Examples: Finding Velocity from Position

Let's apply our knowledge of differentiation to find the velocity of an object given its position as a function of time.

  • Example 1: \(x(t) = At^3 - \dfrac{B}{t}\)

    Here, the position \(x\) is given by the function \(x(t) = At^3 - \dfrac{B}{t}\), where A and B are constants. To find the velocity, we need to differentiate \(x(t)\) with respect to time \(t\):

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} \left( At^3 - \dfrac{B}{t} \right) \)

    Applying the power rule of differentiation:

    \( \qquad v(t) = 3At^2 + \dfrac{B}{t^2} \)

    So, the velocity of the object at any time \(t\) is given by \(3At^2 + \dfrac{B}{t^2}\).

  • Example 2: \(x(t) = A \cos(\omega t)\)

    In this case, the position is given by \(x(t) = A \cos(\omega t)\), where A is the amplitude and \(\omega\) is the angular frequency. Differentiating with respect to time:

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} \left( A \cos(\omega t) \right) \)

    Using the chain rule:

    \( \qquad v(t) = -A\omega \sin(\omega t) \)

    Thus, the velocity of the object at any time \(t\) is \(-A\omega \sin(\omega t)\).

  • Example 3: \(x(t) = Dt^2 + Et + F\)

    Here, the position is given by the quadratic function \(x(t) = Dt^2 + Et + F\), where D, E, and F are constants. Differentiating with respect to time:

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} \left( Dt^2 + Et + F \right) \)

    Applying the power rule:

    \( \qquad v(t) = 2Dt + E \)

    Therefore, the velocity of the object at any time \(t\) is \(2Dt + E\).

  • Example 1: \(v(t) = Gt^2 + Ht\)

    Suppose the velocity \(v\) of an object is given by the function \(v(t) = Gt^2 + Ht\), where G and H are constants. To find the acceleration, we differentiate \(v(t)\) with respect to time \(t\):

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} \left( Gt^2 + Ht \right) \)

    Applying the power rule of differentiation:

    \( \qquad a(t) = 2Gt + H \)

    Thus, the acceleration of the object at any time \(t\) is given by \(2Gt + H\). This means the acceleration changes linearly with time.

  • Example 2: \(v(t) = Ie^{-kt}\)

    In this example, the velocity is given by \(v(t) = Ie^{-kt}\), where I and k are constants. Differentiating with respect to time:

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} \left( Ie^{-kt} \right) \)

    Using the chain rule:

    \( \qquad a(t) = -kIe^{-kt} \)

    So, the acceleration of the object at any time \(t\) is \(-kIe^{-kt}\). The acceleration is also an exponential function and decreases with time.

  • Example 3: \(v(t) = J \cos(\alpha t)\)

    Here, the velocity is given by \(v(t) = J \cos(\alpha t)\), where J is the amplitude and \(\alpha\) is the angular frequency. Differentiating with respect to time:

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} \left( J \cos(\alpha t) \right) \)

    Using the chain rule:

    \( \qquad a(t) = -J\alpha \sin(\alpha t) \)

    Therefore, the acceleration of the object at any time \(t\) is \(-J\alpha \sin(\alpha t)\). The acceleration oscillates sinusoidally with time.

To differentiate more complex functions that often arise in kinematics, we need to understand and apply the fundamental rules of differentiation.

A. Rule of Addition/Subtraction

The derivative of a sum or difference of functions is the sum or difference of their derivatives:

\( \qquad \dfrac{d}{dt} [f(t) \pm g(t)] = \dfrac{df}{dt} \pm \dfrac{dg}{dt} \)

Explanation: This rule states that if you have a function that is the sum or difference of two other functions, you can find its derivative by simply taking the derivatives of each of those functions separately and then adding or subtracting them. This simplifies the differentiation process for complex expressions.

  • Example 1: \(x(t) = t^3 + \sin(t)\)

    Let's say the position of an object is given by \(x(t) = t^3 + \sin(t)\). To find the velocity, we differentiate:

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} (t^3 + \sin(t)) \)

    Applying the rule of addition:

    \( \qquad v(t) = \dfrac{d}{dt} (t^3) + \dfrac{d}{dt} (\sin(t)) = 3t^2 + \cos(t) \)

    So, the velocity is \(v(t) = 3t^2 + \cos(t)\).

  • Example 2: \(v(t) = e^t - t^2\)

    Now, suppose the velocity is given by \(v(t) = e^t - t^2\). To find the acceleration, we differentiate:

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} (e^t - t^2) \)

    Applying the rule of subtraction:

    \( \qquad a(t) = \dfrac{d}{dt} (e^t) - \dfrac{d}{dt} (t^2) = e^t - 2t \)

    Thus, the acceleration is \(a(t) = e^t - 2t\).

  • Example 3: \(a(t) = \cos(t) + t\)

    Let's consider a scenario where acceleration is described by \(a(t) = \cos(t) + t\). To find the rate of change of acceleration (which isn't typically a named kinematic variable, but demonstrates the rule), we differentiate:

    \( \qquad \dfrac{da}{dt} = \dfrac{d}{dt} (\cos(t) + t) \)

    Applying the rule of addition:

    \( \qquad \dfrac{da}{dt} = \dfrac{d}{dt} (\cos(t)) + \dfrac{d}{dt} (t) = -\sin(t) + 1 \)

    Therefore, the rate of change of acceleration is \(-\sin(t) + 1\).

B. Product Rule

The derivative of a product of two functions is given by:

\( \qquad \dfrac{d}{dt} [f(t)g(t)] = f'(t)g(t) + f(t)g'(t) \)

Explanation: This rule tells us that the derivative of two functions multiplied together is the derivative of the first function times the second function, *plus* the first function times the derivative of the second function. It's crucial when differentiating expressions where two functions are multiplied.

  • Example 1: \(x(t) = t \cos(t)\)

    If the position is \(x(t) = t \cos(t)\), then the velocity is:

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} (t \cos(t)) \)

    Applying the product rule:

    \( \qquad v(t) = (1) \cos(t) + t (-\sin(t)) = \cos(t) - t \sin(t) \)

    So, \(v(t) = \cos(t) - t \sin(t)\).

  • Example 2: \(v(t) = t^2 e^{-t}\)

    If the velocity is \(v(t) = t^2 e^{-t}\), then the acceleration is:

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} (t^2 e^{-t}) \)

    Applying the product rule:

    \( \qquad a(t) = (2t) e^{-t} + t^2 (-e^{-t}) = 2te^{-t} - t^2 e^{-t} \)

    Thus, \(a(t) = 2te^{-t} - t^2 e^{-t}\).

  • Example 3: \(a(t) = \sin(t) \cdot t\)

    If acceleration is given by \(a(t) = \sin(t) \cdot t \), then the rate of change of acceleration is:

    \( \qquad \dfrac{da}{dt} = \dfrac{d}{dt} (\sin(t) \cdot t) \)

    Applying the product rule:

    \( \qquad \dfrac{da}{dt} = (\cos(t)) \cdot t + \sin(t) \cdot (1) = t\cos(t) + \sin(t) \)

    Therefore, \(\dfrac{da}{dt} = t\cos(t) + \sin(t)\).

C. Quotient Rule

The derivative of a quotient of two functions is given by:

\( \qquad \dfrac{d}{dt} \left[ \dfrac{f(t)}{g(t)} \right] = \dfrac{f'(t)g(t) - f(t)g'(t)}{[g(t)]^2} \)

Explanation: This rule provides a way to differentiate a function that is expressed as one function divided by another. The derivative involves the derivatives of both the numerator and the denominator, as well as the square of the original denominator.

  • Example 1: \(x(t) = \dfrac{\sin(t)}{t}\)

    If the position is \(x(t) = \dfrac{\sin(t)}{t}\), then the velocity is:

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} \left( \dfrac{\sin(t)}{t} \right) \)

    Applying the quotient rule:

    \( \qquad v(t) = \dfrac{\cos(t) \cdot t - \sin(t) \cdot 1}{t^2} = \dfrac{t \cos(t) - \sin(t)}{t^2} \)

    So, \(v(t) = \dfrac{t \cos(t) - \sin(t)}{t^2}\).

  • Example 2: \(v(t) = \dfrac{t}{t^2 + 1}\)

    If the velocity is \(v(t) = \dfrac{t}{t^2 + 1}\), then the acceleration is:

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} \left( \dfrac{t}{t^2 + 1} \right) \)

    Applying the quotient rule:

    \( \qquad a(t) = \dfrac{(1)(t^2 + 1) - t(2t)}{(t^2 + 1)^2} = \dfrac{1 - t^2}{(t^2 + 1)^2} \)

    Thus, \(a(t) = \dfrac{1 - t^2}{(t^2 + 1)^2}\).

  • Example 3: \(a(t) = \dfrac{e^t}{t}\)

    If the acceleration is given by \(a(t) = \dfrac{e^t}{t}\), then the rate of change of acceleration is:

    \( \qquad \dfrac{da}{dt} = \dfrac{d}{dt} \left( \dfrac{e^t}{t} \right) \)

    Applying the quotient rule:

    \( \qquad \dfrac{da}{dt} = \dfrac{e^t \cdot t - e^t \cdot 1}{t^2} = \dfrac{e^t(t - 1)}{t^2} \)

    Therefore, \(\dfrac{da}{dt} = \dfrac{e^t(t - 1)}{t^2}\).

D. Chain Rule

The derivative of a composite function is given by:

\( \qquad \dfrac{d}{dt} [f(g(t))] = f'(g(t)) \cdot g'(t) \)

Explanation: This rule is used when differentiating a function within another function (a composite function). It states that you take the derivative of the outer function, evaluated at the inner function, and then multiply by the derivative of the inner function.

  • Example 1: \(x(t) = \sin(t^2)\)

    If the position is \(x(t) = \sin(t^2)\), then the velocity is:

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} (\sin(t^2)) \)

    Applying the chain rule:

    \( \qquad v(t) = \cos(t^2) \cdot (2t) = 2t \cos(t^2) \)

    So, \(v(t) = 2t \cos(t^2)\).

  • Example 2: \(v(t) = \cos(e^t)\)

    If the velocity is \(v(t) = \cos(e^t)\), then the acceleration is:

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} (\cos(e^t)) \)

    Applying the chain rule:

    \( \qquad a(t) = -\sin(e^t) \cdot (e^t) = -e^t \sin(e^t) \)

    Thus, \(a(t) = -e^t \sin(e^t)\).

  • Example 3: \(a(t) = (t^3 + 1)^5\)

    If the acceleration is given by \(a(t) = (t^3 + 1)^5\), then the rate of change of acceleration is:

    \( \qquad \dfrac{da}{dt} = \dfrac{d}{dt} ((t^3 + 1)^5) \)

    Applying the chain rule:

    \( \qquad \dfrac{da}{dt} = 5(t^3 + 1)^4 \cdot (3t^2) = 15t^2(t^3 + 1)^4 \)

    Therefore, \(\dfrac{da}{dt} = 15t^2(t^3 + 1)^4\).

  • Example 4: \(x(t) = e^{\sin(t)}\)

    If the position is \(x(t) = e^{\sin(t)}\), then the velocity is:

    \( \qquad v(t) = \dfrac{dx}{dt} = \dfrac{d}{dt} (e^{\sin(t)}) \)

    Applying the chain rule:

    \( \qquad v(t) = e^{\sin(t)} \cdot (\cos(t)) = \cos(t) e^{\sin(t)} \)

    So, \(v(t) = \cos(t) e^{\sin(t)}\).

  • Example 5: \(v(t) = \ln(t^2 + 1)\)

    If the velocity is \(v(t) = \ln(t^2 + 1)\), then the acceleration is:

    \( \qquad a(t) = \dfrac{dv}{dt} = \dfrac{d}{dt} (\ln(t^2 + 1)) \)

    Applying the chain rule:

    \( \qquad a(t) = \dfrac{1}{t^2 + 1} \cdot (2t) = \dfrac{2t}{t^2 + 1} \)

    Thus, \(a(t) = \dfrac{2t}{t^2 + 1}\).

Integration is the reverse process of differentiation. In kinematics, it allows us to determine displacement from velocity and velocity from acceleration, providing a complete picture of an object's motion.

Introduction to Integration

Integration, often called "anti-differentiation," is a core concept in calculus. While differentiation tells us the *rate of change* of a function, integration allows us to find the *original function* when we know its rate of change. This is incredibly useful in physics, especially in kinematics.

Displacement as Integral of Velocity:

If we know the velocity of an object as a function of time, we can find its displacement (change in position) by integrating the velocity function:

\( \qquad x(t) = \int v(t) \, dt \)

This integral represents the area under the velocity-time curve, which gives us the total displacement of the object over a given time interval.

Velocity as Integral of Acceleration:

Similarly, if we know the acceleration of an object as a function of time, we can find its velocity by integrating the acceleration function:

\( \qquad v(t) = \int a(t) \, dt \)

This integral represents the area under the acceleration-time curve, which gives us the total change in velocity of the object over a given time interval.

Understanding the Constant of Integration: The Missing Piece

When we perform an indefinite integral (one without specific limits), we always get a "constant of integration," often denoted as \(C\). This constant represents the unknown initial value of the function we're trying to find.

In kinematics:

  • When finding velocity from acceleration, \(C\) represents the *initial velocity* \(v(0)\).
  • When finding position from velocity, \(C\) represents the *initial position* \(x(0)\).

Therefore, the complete equations are:

\( \qquad v(t) = \int a(t) \, dt + v(0) \)

\( \qquad x(t) = \int v(t) \, dt + x(0) \)

It's crucial to use the given initial conditions to determine the value of \(C\) to get the correct velocity or position function.

Illustrative Examples:
  • A. Finding Velocity from Acceleration: \(a(t) = At\), \(v(0) = v_0\)

    Suppose the acceleration of an object is given by \(a(t) = At\), where \(A\) is a constant, and its initial velocity at \(t = 0\) is \(v_0\). To find the velocity \(v(t)\), we integrate the acceleration function:

    \( \qquad v(t) = \int a(t) \, dt = \int At \, dt \)

    Applying the power rule of integration:

    \( \qquad v(t) = \dfrac{1}{2}At^2 + C \)

    To find the constant of integration \(C\), we use the initial condition \(v(0) = v_0\):

    \( \qquad v_0 = \dfrac{1}{2}A(0)^2 + C \Rightarrow C = v_0 \)

    Therefore, the velocity function is:

    \( \qquad v(t) = \dfrac{1}{2}At^2 + v_0 \)

  • B. Finding Velocity from Acceleration: \(a(t) = B \sin(\omega t)\), \(v(0) = 0\)

    Consider an object with acceleration \(a(t) = B \sin(\omega t)\), where \(B\) and \(\omega\) are constants, and initial velocity \(v(0) = 0\). Integrating to find velocity:

    \( \qquad v(t) = \int a(t) \, dt = \int B \sin(\omega t) \, dt \)

    Using the integral of \(\sin(t)\):

    \( \qquad v(t) = -\dfrac{B}{\omega} \cos(\omega t) + C \)

    Applying the initial condition \(v(0) = 0\):

    \( \qquad 0 = -\dfrac{B}{\omega} \cos(0) + C \Rightarrow C = \dfrac{B}{\omega} \)

    Thus, the velocity function is:

    \( \qquad v(t) = -\dfrac{B}{\omega} \cos(\omega t) + \dfrac{B}{\omega} = \dfrac{B}{\omega} (1 - \cos(\omega t)) \)

  • C. Finding Velocity from Acceleration: \(a(t) = C\), \(v(0) = u\)

    An object moves with constant acceleration \(a(t) = C\) and has an initial velocity \(v(0) = u\). Integrating to find velocity:

    \( \qquad v(t) = \int a(t) \, dt = \int C \, dt \)

    The integral of a constant is:

    \( \qquad v(t) = Ct + C' \)

    Applying the initial condition \(v(0) = u\):

    \( \qquad u = C(0) + C' \Rightarrow C' = u \)

    Therefore, the velocity function is:

    \( \qquad v(t) = Ct + u \)

Illustrative Examples:
  • A. Finding Position from Velocity: \(v(t) = Dt^2\), \(x(0) = x_0\)

    Suppose the Velocity of an object is given by \(v(t) = Dt^2\), where \(D\) is a constant, and its initial position at \(t = 0\) is \(x_0\). To find the position \(x(t)\), we integrate the Velocity function:

    \( \qquad x(t) = \int v(t) \, dt = \int Dt^2 \, dt \)

    Applying the power rule of integration:

    \( \qquad x(t) = \dfrac{1}{3}Dt^3 + C \)

    To find the constant of integration \(C\), we use the initial condition \(x(0) = x_0\):

    \( \qquad x_0 = \dfrac{1}{3}D(0)^3 + C \Rightarrow C = x_0 \)

    Therefore, the position function is:

    \( \qquad x(t) = \dfrac{1}{3}Dt^3 + x_0 \)

  • B. Finding Position from Velocity: \(v(t) = E e^{-kt}\), \(x(0) = 0\)

    Consider an object with Velocity \(v(t) = E e^{-kt}\), where \(E\) and \(k\) are constants, and initial position \(x(0) = 0\). Integrating to find position:

    \( \qquad x(t) = \int v(t) \, dt = \int E e^{-kt} \, dt \)

    Using the integral of \(e^t\):

    \( \qquad x(t) = -\dfrac{E}{k} e^{-kt} + C \)

    Applying the initial condition \(x(0) = 0\):

    \( \qquad 0 = -\dfrac{E}{k} e^{-k(0)} + C \Rightarrow C = \dfrac{E}{k} \)

    Thus, the Velocity function is:

    \( \qquad x(t) = -\dfrac{E}{k} e^{-kt} + \dfrac{E}{k} \)

  • C. Finding Velocity from Acceleration: \(v(t) = F cos(\alpha t)\), \(x(0) = s_0\)

    An object moves with cosinusoidial Velocity \(v(t) = F cos(\alpha t)\) and has an initial position \(x(0) = s_0\). Integrating to find position:

    \( \qquad x(t) = \int v(t) \, dt = \int F cos(\alpha t) \, dt \)

    The integral of cosine is:

    \( \qquad x(t) = \dfrac{F}{\alpha}sin(\alpha t) + C' \)

    Applying the initial condition \(x(0) = s_0\):

    \( \qquad s_0 = \dfrac{F}{\alpha}sin(\alpha(0)) + C' \Rightarrow C' = s_0 \)

    Therefore, the position function is:

    \( \qquad x(t) = \dfrac{F}{\alpha}sin(\alpha t) + s_0 \)

Integration by substitution, also known as u-substitution, is a powerful technique for simplifying integrals by changing the variable of integration. It's essentially the reverse of the chain rule in differentiation.

General Explanation:

The goal of u-substitution is to transform a complicated integral into a simpler one that we can directly evaluate using standard formulas. The process involves the following steps:

  1. Choose a substitution: Identify a part of the integrand (the function being integrated) that can be replaced with a new variable, \(u\). This is often a function whose derivative also appears in the integrand (or can be made to appear with some algebraic manipulation).
  2. Find \(du\): Calculate the derivative of \(u\) with respect to \(t\) (or whatever the original variable is), and express it as \(du = \dfrac{du}{dt} \, dt\).
  3. Substitute: Replace the original expression in the integral with expressions involving \(u\) and \(du\). The goal is to get an integral that is entirely in terms of \(u\).
  4. Integrate: Evaluate the new integral with respect to \(u\).
  5. Substitute back: Replace \(u\) with its original expression in terms of \(t\) to get the final answer.

Mathematically, if we can write the integral in the form \(\int f(g(t)) \cdot g'(t) \, dt\), then we can use the substitution \(u = g(t)\) and \(du = g'(t) \, dt\) to transform the integral into \(\int f(u) \, du\), which is hopefully easier to evaluate.

Illustrative Examples:
  • Example 1: \(\int 2t \cos(t^2) \, dt\)

    Let's evaluate the integral \(\int 2t \cos(t^2) \, dt\) using substitution.

    1. Choose a substitution: Let \(u = t^2\).
    2. Find \(du\): Then \(\dfrac{du}{dt} = 2t\), so \(du = 2t \, dt\).
    3. Substitute: The integral becomes \(\int \cos(u) \, du\).
    4. Integrate: \(\int \cos(u) \, du = \sin(u) + C\).
    5. Substitute back: Replace \(u\) with \(t^2\) to get \(\sin(t^2) + C\).

    Therefore, \(\int 2t \cos(t^2) \, dt = \sin(t^2) + C\).

  • Example 2: \(\int e^{\sin(t)} \cos(t) \, dt\)

    Let's evaluate the integral \(\int e^{\sin(t)} \cos(t) \, dt\) using substitution.

    1. Choose a substitution: Let \(u = \sin(t)\).
    2. Find \(du\): Then \(\dfrac{du}{dt} = \cos(t)\), so \(du = \cos(t) \, dt\).
    3. Substitute: The integral becomes \(\int e^u \, du\).
    4. Integrate: \(\int e^u \, du = e^u + C\).
    5. Substitute back: Replace \(u\) with \(\sin(t)\) to get \(e^{\sin(t)} + C\).

    Therefore, \(\int e^{\sin(t)} \cos(t) \, dt = e^{\sin(t)} + C\).

  • Example 3: \(\int \dfrac{t}{t^2 + 1} \, dt\)

    Let's evaluate the integral \(\int \dfrac{t}{t^2 + 1} \, dt\) using substitution.

    1. Choose a substitution: Let \(u = t^2 + 1\).
    2. Find \(du\): Then \(\dfrac{du}{dt} = 2t\), so \(du = 2t \, dt\). This means \(t \, dt = \dfrac{1}{2} \, du\).
    3. Substitute: The integral becomes \(\int \dfrac{1}{2u} \, du = \dfrac{1}{2} \int \dfrac{1}{u} \, du\).
    4. Integrate: \(\dfrac{1}{2} \int \dfrac{1}{u} \, du = dfrac{1}{2} \ln|u| + C\).
    5. Substitute back: Replace \(u\) with \(t^2 + 1\) to get \(\dfrac{1}{2} \ln(t^2 + 1) + C\).

    Therefore, \(\int \dfrac{t}{t^2 + 1} \, dt = \dfrac{1}{2} \ln(t^2 + 1) + C\).

  • Example 4: \(\int (4t+3)^5 dt\)

    Let's evaluate the integral \(\int (4t+3)^5 dt\) using substitution.

    1. Choose a substitution: Let \(u = 4t+3\).
    2. Find \(du\): Then \(\dfrac{du}{dt} = 4\), so \(du = 4 dt\). This means \(dt = \dfrac{1}{4} \, du\).
    3. Substitute: The integral becomes \(\int u^5 \dfrac{1}{4} \, du = \dfrac{1}{4} \int u^5 \, du\).
    4. Integrate: \(\dfrac{1}{4} \int u^5 \, du = dfrac{1}{4}\dfrac{u^6}{6} + C\).
    5. Substitute back: Replace \(u\) with \(4t+3\) to get \(\dfrac{(4t+3)^6}{24} + C\).

    Therefore, \(\int (4t+3)^5 dt = \dfrac{(4t+3)^6}{24} + C\).

  • Example 5: \(\int \dfrac{1}{\sqrt{1-4t^2}} dt\)

    Let's evaluate the integral \(\int \dfrac{1}{\sqrt{1-4t^2}} dt\) using substitution.

    1. Choose a substitution: Let \(u = 2t\).
    2. Find \(du\): Then \(\dfrac{du}{dt} = 2\), so \(du = 2 dt\). This means \(dt = \dfrac{1}{2} \, du\).
    3. Substitute: The integral becomes \(\int \dfrac{1}{\sqrt{1-u^2}} \dfrac{1}{2} \, du = \dfrac{1}{2}\int \dfrac{1}{\sqrt{1-u^2}} \, du\).
    4. Integrate: \(\dfrac{1}{2}\int \dfrac{1}{\sqrt{1-u^2}} \, du = dfrac{1}{2}arcsin(u) + C\).
    5. Substitute back: Replace \(u\) with \(2t\) to get \(\dfrac{1}{2}arcsin(2t) + C\).

    Therefore, \(\int \dfrac{1}{\sqrt{1-4t^2}} dt = \dfrac{1}{2}arcsin(2t) + C\).
